## :running: Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Super-Chama/js-boomerang.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Serve the app using npm
   ```sh
   npm run start
   ```
